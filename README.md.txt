# Basic Gates using Verilog

# Project Overview

In this project, I implemented basic digital logic gates using Verilog HDL and verified their functionality using a SystemVerilog testbench in Vivado.

 Gates Implemented

* AND Gate
* OR Gate
* NOT Gate
* NAND Gate
* NOR Gate
* XOR Gate

 📂 Files Included

* `basic_gates.v` → Verilog code for implementing all basic gates
* `tb_basic_gates.sv` → SystemVerilog testbench used for simulation
* `waveform.png` → Output waveform obtained after simulation

 🛠 Tools Used

* Xilinx Vivado
* Verilog (for RTL design)
* SystemVerilog (for testbench)

 ▶️ Simulation Details

I applied all possible input combinations (00, 01, 10, 11) to verify the correctness of each logic gate. The outputs were observed using the waveform viewer in Vivado.

 📊 Result

All the logic gates produced correct outputs as per their truth tables.

 Learning Outcome

* Understood basic digital logic design using Verilog
* Learned how to write and run a testbench
* Got hands-on experience with Vivado simulation
* Understood the difference between Verilog and SystemVerilog
